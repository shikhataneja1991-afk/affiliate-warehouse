AFFILIATE WAREHOUSE — Website Files
=====================================

FOLDER STRUCTURE:
-----------------
affiliate-warehouse/
├── index.html        ← MAIN PAGE (open this in browser)
├── about.html        ← About page (placeholder)
├── service.html      ← Services page (placeholder)
├── team.html         ← Team page (placeholder)
├── why.html          ← Why page (placeholder)
├── css/
│   └── style.css     ← All styles (edit colours here)
├── js/
│   └── main.js       ← Animations & form interactions
├── fonts/            ← Add any local font files here
├── images/           ← Add logo & images here
└── README.txt        ← This file

HOW TO EDIT:
------------
• Change accent colour:  css/style.css → line 8 → --accent: #A78BFA
• Change background:     css/style.css → line 6 → --bg: #0B111A
• Edit page text:        index.html (search for the text you want to change)
• Add your logo:         Replace "AffiliateWarehouse" logo text in index.html
• Add your ABN:          Search for "[Your ABN here]" in index.html

TO USE ON WORDPRESS:
--------------------
1. Upload entire folder to your hosting via FTP or cPanel File Manager
2. Or use a plugin like "WP File Manager" to upload
3. Set index.html as your homepage

TO OPEN LOCALLY:
----------------
Double-click index.html — opens directly in any browser. No server needed.
